# Veterinary Clinic API

REST API для ветеринарной клиники, разработанный на Django REST Framework и PostgreSQL.

## Описание проекта

Проект предоставляет API для управления ветеринарной клиникой:

- владельцы животных
- питомцы
- врачи
- записи на приём

API позволяет выполнять CRUD-операции:
- создание
- просмотр
- обновление
- удаление данных

# База данных и модели в приложении api_v1

    Содержание: Проектирование объектной структуры данных на примере сущностей цифровой библиотеки: Author, Book, Genre и Comment. Демонстрируется синтаксис связей «один ко многим» (ForeignKey с каскадным удалением) и «многие ко многим» (ManyToManyField).

# Сериализаторы (serializers.py)

    Содержание: Разбор роли сериализаторов как «работников склада», трансформирующих сложные ORM-объекты Python в JSON-структуры и обратно. Приводится пример реализации AuthorSerializer, а также продвинутые приёмы на примере BookSerializer: разделение потоков чтения и записи данных (комбинация read_only=True для вложенного объекта автора и write_only=True через PrimaryKeyRelatedField для корректной записи по author_id).
